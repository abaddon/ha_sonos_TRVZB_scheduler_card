SVG Chart Test - Schedule Graph View

This page tests if the SVG chart renders correctly in the browser.

What You Should See:

  • Grid lines (horizontal and vertical, dashed, gray)
  • X-axis (horizontal line at bottom) with time labels (0:00 to 24:00)
  • Y-axis (vertical line at left) with temperature labels (4°C to 29°C)
  • Axis titles ("Time of Day" and "Temperature")
  • Cyan/blue line connecting temperature points
  • 5 colored circular points with white borders
  • Temperature and time labels above/below each point

If Nothing Appears:

  • Open browser console (F12) and check for errors
  • Check if SVG viewBox is working correctly
  • Verify CSS is not hiding elements
